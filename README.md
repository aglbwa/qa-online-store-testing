# Портфолио QA

## Проект

Тестирование веб-приложения SauceDemo.

Цель проекта — продемонстрировать навыки ручного тестирования веб-приложений.

---

## Что было протестировано

- Авторизация
- Каталог товаров
- Сортировка товаров
- Карточка товара
- Корзина
- Оформление заказа
- Навигация

---

## Виды тестирования

- Функциональное
- Smoke Testing
- Регрессионное
- UI-тестирование
- Исследовательское
- Тестирование совместимости
- Негативное тестирование
- Граничные значения

---

## Документация

- Стратегия тестирования
- Тест-кейсы
- Чек-листы

- Итоговый отчет
